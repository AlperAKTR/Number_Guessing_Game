** Number Guessing Game **
----------------------------

A simple command-line number guessing game written in Python.


Features
----------------------------

- Random number generation
- Text input validation
- Feedback: too high / too low
- Attempt counter
- Play again option

Gameplay Example
----------------------------

-Welcome to the Number Guessing Game!
-I will think of a number between 1 and 50, and your job is to guess it!

- Do you want to play? (y/n): y
- Great! Let's begin. I've chosen my number.
- Enter your guess (1-50): 25
- Too low!
- Enter your guess (1-50): 40
- Too high!
- Enter your guess (1-50): 33
- Correct! The number was 33.
- You guessed it in 3 attempts.
